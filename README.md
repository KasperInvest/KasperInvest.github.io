# kasperneslarsen.github.io

Personlig analyseportefølje og equity research-nettside av Kasper Nes Larsen.

Publisert via GitHub Pages på: **https://kasperneslarsen.github.io**

## Innhold
- Aksjeanalyser med verdivurdering og investeringsanbefalinger
- Kortere markedsnotater og metodikk-innlegg
- Om meg og kontaktside

## Teknologi
Ren HTML/CSS/JS — ingen rammeverk, ingen byggsteg. Bare last opp filene og det virker.

## Legg til ny rapport
1. Lagre rapporten som `selskapsnavn-rapport.html` i rotmappen
2. Legg til et rapportkort i `index.html` under `#page-research`
3. Push til GitHub — siden oppdateres automatisk
